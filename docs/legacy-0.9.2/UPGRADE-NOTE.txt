Anchor Memory 0.9.2 FINAL — 长聊天、召回、事务与面板语义修正版

安装：
1. 在 SillyTavern 的扩展管理中移除或覆盖旧版 Anchor Memory 文件夹。
2. 安装本压缩包，确保扩展目录内直接包含 manifest.json、index.js、settings.html、style.css。
3. 重启或刷新 SillyTavern；扩展面板应显示版本 0.9.2。

升级行为：
- 旧聊天记忆保留并自动迁移到数据版本 9。
- 没有原生消息 ID 的旧聊天会生成并持久化插件稳定消息键；旧摘要、锚点、合并和关系历史会同步重映射，不会因为删除中间楼层而让后续楼层整体漂移。
- 0.9.1 中隐式默认开启的“额外旧楼动态召回”会在升级时恢复为默认关闭。升级后用户手动开关一次，即视为明确选择。
- 自定义提示词会尽量保留；只有被识别为旧版内嵌人物/物品表结构的预设会迁移为新版格式。

默认主模型输入：
- 始终保留最近 3 个 AI 回合原文（可调）。
- 发送一个顺序固定、总长度受限的六段记忆系统块。
- 动态旧楼召回默认关闭；人物名称匹配和重要物品账本是独立开关。

建议升级后先打开“召回”页查看完整拼接预览，再进行一次正常生成。
