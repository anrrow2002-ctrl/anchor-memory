# Anchor Memory 0.9.14 升级说明

1. 在“工具”页导出一次当前记忆 JSON 作为额外备份。
2. 完全关闭 SillyTavern 页面。
3. 用本包中的 `anchor-memory-0.9.11-REWRITE-CONTROLS` 整个目录覆盖原同名目录，不要把 0.9.14 安装成第二个并存扩展。
4. 重新打开页面，确认设置页显示版本 0.9.14。
5. 手机端重新点击“拉取副 API 模型”：模型栏会先清空，再读取当前地址和密钥；拉取失败时请按界面显示的代理错误和直连错误排查。

本版不改变聊天记忆数据结构。已有逐楼摘要、锚点、累计合并、关系表、人物、物品、场景和归档不会因覆盖安装被删除。

注意：归档整理版只能加载到其他新开场，不能加载回它的原聊天；这是为了防止精简转档副本覆盖原聊天的详细记忆。
