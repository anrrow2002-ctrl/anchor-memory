# Upgrade to 0.9.17

1. Replace the existing `anchor-memory-0.9.11-REWRITE-CONTROLS` directory with this package.
2. Refresh SillyTavern.
3. Open Anchor Memory → Settings → Secondary API.
4. Fill the current connection, enter a preset name, and choose **Save as new preset**.
5. Select any saved name later to restore its URL, key, model, and model list immediately.

No chat-memory reset is required. Godlogs, anchors, cumulative merges, entity indexes, hidden-message ownership, and archives remain unchanged.
