# 升级到 Anchor Memory 0.9.15

1. 关闭或暂停当前正在运行的摘要、锚点、合并与索引任务。
2. 用压缩包中的 `anchor-memory-0.9.11-REWRITE-CONTROLS` 整个目录覆盖旧同名目录。
3. 不要同时保留 0.9.14 与 0.9.15 两份扩展目录。
4. 刷新 SillyTavern 页面并打开原聊天。
5. 原聊天数据会自动迁移；无需重置记忆，也不要删除聊天 metadata。

升级后，旧的 `secondary-timeout` 状态会被转换为可读说明并清除旧重试锁。人物索引会按最多 10 楼/9,000 字符一段继续安全重建。若同一段连续三次失败，插件会暂停自动重试；检查副 API 后，在“人物动态”页手动继续即可。
