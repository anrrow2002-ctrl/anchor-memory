# 升级到 Anchor Memory 0.9.9

直接覆盖安装即可，不需要删除旧插件，也不需要清空聊天 metadata。

本版不改变现有记忆数据结构。逐楼摘要、分段锚点、累计历史、人物关系、人物纪要、人物库、物品、场景、向量引用、API 配置与档案都会保留。

## 一键暂停

打开锚点书后，标题栏右上角有“暂停插件 / 启动插件”按钮；SillyTavern 的扩展设置列表中也有同一按钮。

暂停后：

- 不向主模型注入 Anchor Memory；
- 不隐藏旧楼，并恢复此前由本插件管理的隐藏楼层；
- 不生成逐楼摘要、锚点或累计合并；
- 不执行动态召回或新的后台 API 请求；
- 保留全部数据与设置。

再次点击“启动插件”后，插件会重新同步当前聊天、恢复最近原文窗口规则，并继续处理尚未完成的记忆任务。
