# Anchor Memory 0.9.12 升级说明

## 档案流程

1. 在原聊天的“档案”页输入档案名，点击“保存当前快照”。
2. 档案卡片会显示“补齐摘要并全量合并”。
3. 点击后，插件会补齐所有缺失/失效逐楼摘要，并把不足常规合并周期的尾段立即并入最终累计全量记忆。
4. 整理完成后，档案状态变为“已全量整理”，此时才可加载到新开场。

## 转档内容

整理版只携带：

- 最后一份累计全量合并；
- 人物关系表；
- 人物纪要与人物库；
- 物品与场景状态；
- 当前剧情时间/地点等状态。

不会携带旧逐楼摘要、旧分段锚点和旧楼层编号，因此新开场从第1楼重新计数时不会出现两个“第1楼”。

## 兼容说明

旧式未整理快照不会再直接加载。请回到原聊天重新保存并完成全量整理；这是为了避免把未合并尾段和旧楼层编号直接注入新档。
